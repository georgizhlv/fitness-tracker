# Тестване на Фитнес Тракер

## 🎯 Два начина за използване

### 1️⃣ **Автоматична демонстрация (по подразбиране)**
Показва всички функционалности с предварително определени данни.

```bash
.\build.ps1 -Run
```

**Демонстрира:**
- ✅ Добавяне на 5 различни тренировки
- ✅ Потребителски профил с BMI
- ✅ История на тренировките сортирана по дата
- ✅ Статистика и средни показатели
- ✅ Филтриране по тип тренировка
- ✅ Целево проследяване
- ✅ Разпределение на калории
- ✅ Персистентност (запис и прочит от файл)

---

### 2️⃣ **Интерактивна демонстрация (НОВА!)**
Интерактивно тестване - вие контролирате всяка стъпка.

```bash
.\build_interactive.ps1 -Interactive
```

**Функции:**
1. **Добавяне на тренировка (Кардио)** — Въведете дата, продължителност, пулс, интензивност
2. **Добавяне на тренировка (Силова)** — Серии, повторения, мускулна група
3. **Добавяне на тренировка (Йога)** — Стил (Hatha/Vinyasa/Yin), време на ден
4. **Добавяне на тренировка (Плуване)** — Дължини, стил на плуване
5. **Преглед на история** — Вижте всички добавени тренировки
6. **Преглед на статистика** — Общо показатели
7. **Потребителски профил** — BMI и данни
8. **Филтриране по тип** — Покажи само определен тип тренировка
9. **Цел и прогрес** — Дали сте достигнали целите си
10. **Средни показатели** — Средна продължителност и калории
11. **Разпределение на калории** — По вид тренировка
12. **Сортиране по дата** — Наредете историята хронологично
13. **Изтриване на тренировка** — Премахнете тренировка по индекс
14. **Запазване и прочит** — Работа с файлове

## 🔧 Компилиране

### Нормален режим (демонстрация)
```bash
.\build.ps1
```

### С запуск на програмата
```bash
.\build.ps1 -Run
```

### Интерактивен режим
```bash
.\build_interactive.ps1 -Interactive
```

### Почистване
```bash
.\build.ps1 -Clean
```

---

## 📝 Формат на входа

### За дата: `YYYY-MM-DD`
- Пример: `2025-06-01`

### За целочисла (минути, пулс, серии)
- Пример: `45`, `150`, `4`

### За текст (интензивност, стил)
- Пример: `интервали`, `Hatha`, `бруст`

## 💾 Файлови операции

- Запазване на текущите тренировки
- Прочит на запазени тренировки
